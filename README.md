
## CustomerApi

Api which list,Store,Edit,Delete and Update Customers.

## Requirements

Docker

## Installation 

- Clone repository so that you can work on it 
- Run `sudo docker-compose up`

## API Endpoints

    * `GET /Customers`
        * Show all Customers
        
    * `GET /Customers/{id}`
        * show customer details id.
        
    * `POST /customers`
        * Add customer data
        
    * `DELETE /Customers/{id}`
            * Delete requested Customer
            
    * `PUT /Customers/{id}`
            * Update customer data
